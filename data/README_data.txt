MongoImport
===========
All of the starting data used in this project is imported to mongo using MongoImport.

Sample user data is stored in .json form, and can be imported using the command: "mongoimport --jsonArray"

Sample movie data is stored in .csv form, and can be imported by specifying .csv as --type.
